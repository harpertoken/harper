# Governance

This document outlines the governance model for the Harper project.

## Roles

- **Maintainers**: Responsible for project direction, code review, and releases. See [MAINTAINERS.md](MAINTAINERS.md) for a list of current maintainers.
- **Contributors**: Anyone who contributes code or documentation.

## Decision Making

Major decisions are made by maintainers via consensus.
