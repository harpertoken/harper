# Documentation

## Serve Locally
- `mkdocs serve`

## Build
- `mkdocs build`

## Deploy (on main)
- `mkdocs gh-deploy`

## Check Links
- `mkdocs build --strict`

## Key Docs
- `docs/` - documentation files
- `website/` - website source
- `mkdocs.yml` - mkdocs configuration
