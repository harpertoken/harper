How it works

You describe what you want in plain text. Harper generates shell commands, shows them to you, and waits for approval before running anything.

Features

  • Preview every command before execution
  • Full audit trail of all sessions
  • Sandbox isolation (bubblewrap/linux, sandbox-exec/macos)
  • Works with OpenAI, Gemini, SambaNova, or Ollama

Sandbox

Commands run in isolation. On Linux, bubblewrap creates a minimal namespace. On macOS, sandbox-exec provides App Sandbox profiles. No network unless you allow it.
