How it works

You describe what you want in plain text. Harper can inspect files, edit code, run commands, and keep track of multi-step work while still showing you the concrete actions it plans to take.

Features

  • Preview commands before execution
  • Approval gates for sensitive operations
  • Session history with readable titles, plan tracking, and audit logs
  • Works with OpenAI, Gemini, SambaNova, Cerebras, or Ollama

Sandbox

Commands can run in isolation. On Linux, bubblewrap creates a minimal namespace. On macOS, sandbox-exec provides App Sandbox profiles. Networked or higher-risk actions can stay behind explicit approval.
